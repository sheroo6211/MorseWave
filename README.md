# MorseWave
Learn Morse Code  Beginner-friendly Morse code learning app. No login or installation needed. Works in any browser.  ## Features - Step-by-step lessons with sound - 5 practice modes - XP, levels &amp; achievement badges - Text ↔ Morse converter - Dark mode &amp; adjustable speed  ## Live App 👉 [Open MorseWave](https://sheroo6211.github.io/morsewave)
