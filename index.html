What Is Morse Code?

Morse code is a system for sending messages using only two signals: a short dot (·) and a long dash (—). Every letter, number and punctuation mark has its own unique pattern of dots and dashes. Invented in the 1830s, it was used to send telegraphs across the world long before phones existed!

💡

Why Learn It?

Morse code is still used today — by pilots, sailors, and ham radio operators. It's also a brilliant brain exercise for pattern recognition. Learning it sharpens your memory, attention span, and auditory processing. Plus it's really cool — you can communicate in secret!

🧠

How This App Works

We teach letters in small groups, not all at once. Each session builds on the last. You'll hear sounds, see animations, complete quizzes, and earn XP as you go. The app uses spaced repetition — reviewing things just before you'd forget them for maximum retention.

Timing Rules

· Dot = 1 unit
— Dash = 3 units
Gap between symbols = 1 unit
Gap between letters = 3 units
Gap between words = 7 units
These rules keep the rhythm consistent so anyone can understand!
